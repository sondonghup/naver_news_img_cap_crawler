### 네이버 뉴스를 보면 뉴스 관련된 사진과 바로 밑에 그에 대한 설명이 있는 것을 볼 수 있다.

### 이 두가지를 잘 긁어오면 이미지 캡셔닝 데이터로 잘 사용 할 수 있지 않을까

***

### 뉴스 카테고리와 시작 날짜 끝 날짜를 입력하면 그 기간 안의 해당 카테고리 뉴스의 이미지와 캡션을 크롤링 합니다.
- ### 뉴스 카테고리
```
    경제 : Economy
    정치 : Politics
    사회 : Society
    생활 : LifeCulture
    세계 : World
    IT : It
    사설칼럼 : Opinion
```
- ### 사용법
```
$ ㅔ
```
